# Logging

| Name                               | Description                                                           |
|:-----------------------------------|:----------------------------------------------------------------------|
|[Log](../options/log.md)  | Gets or sets an action to `log` all database events as soon as they happen.                 |
|[UseLogDump](../options/use-log-dump.md)  | Gets or sets if all `log` related to database event should be stored in a `LogDump` properties. |
|[LogDump](../options/log-dump.md)  | Gets all `logged` database event when `UseLogDump` is enabled.                |
